## Installation

1. Download or clone this extension folder
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked"
5. Select the `udemy-speed-controller` folder
6. The extension is now installed!

## Usage

1. Navigate to any Udemy course lecture page
2. Click the extension icon in your Chrome toolbar
3. Use the slider or preset buttons to set your desired playback speed
4. The speed will be applied immediately and saved for future lectures

## Speed Range

- **Minimum**: 0.25x (quarter speed)
- **Maximum**: 6x (six times speed)
- **Default**: 1x (normal speed)

## Preset Speeds

Quick access buttons for: 0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x, 2.5x, 3x, 4x, 6x

## Notes

- The extension only works on udemy.com pages
- If you see "Please refresh the Udemy page", simply refresh the page and try again
- Your preferred speed is automatically saved and will persist across browser sessions
- When you navigate to a new lecture, the extension will automatically apply your saved speed

## Troubleshooting

**Extension not working?**
- Make sure you're on a Udemy page with a video
- Try refreshing the page
- Check that the extension is enabled in `chrome://extensions/`

**Speed not applying?**
- Wait for the video to load completely
- Try clicking a preset button instead of using the slider
- Refresh the page and try again

## Privacy

This extension:
- Only runs on udemy.com pages
- Stores your speed preference locally in your browser
- Does not collect or transmit any data
- Does not require any special permissions beyond accessing Udemy pages
